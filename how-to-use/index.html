How to Use SCQ Scoreboard

SCQ Scoreboard uses a dual-window setup: a Control Panel for the coach/quizmaster and a Main Scoreboard for the audience.

A) How to Set Up

The School’s Challenge Quiz Scoreboard works best on a PC or laptop connected to a second screen (preferably wired).

  1. Connect the second screen to the PC/laptop (typically via HDMI).
  2. Set the projection mode to Extend.
  3. (Projection modes include PC screen only, Duplicate, Extend, and Second screen only.)
  4. Open the platform.
  5. Drag the Main Scoreboard window to the second screen.
  6. The second screen is an extension of the main display.
  7. Keep the Control Panel on the main PC/laptop screen for the coach.
  8. Teams will see only the Main Scoreboard, while the coach manages scores, timers, visuals, and audio from the Control Panel.

    9. See the section on "Presenting mode" for a quicker way to set up the scoreboard. Premium users only

B) How to Use

1) Settings Menu

Team Names

  • Click a team name or the dropdown arrow to select a school from the built-in list, or manually type a custom name and press Update Names.

Select Section

  • Choose the active section using the Settings menu or by clicking directly on the round names in the Scoreboard Tracker.
  • Sections: Alternate, Speed Minutes 1–3, or Buzzer.

Reset Match

  • Resets scores, timers, trackers, queries, visuals, and audio.
  • A confirmation prompt prevents accidental resets.

2) Main Control Panel

Scoreboard Tracker

  • Displays a full breakdown of questions answered in each section.

Query Buttons (qry− / qry+)

  • Apply manual score adjustments per section.
  • Alternate & Speed: ±1 point
  • Buzzer: ±4 points
  • Each query is logged visually in the row until the match is reset.

Scoring Buttons

  • Correct/Wrong buttons update scores based on the active section.
  • Keyboard shortcuts are available for faster scoring:
  • 1 — Team A Correct
  • 2 — Team A Wrong
  • 8 — Team B Correct
  • 9 — Team B Wrong
  • Undo last score: Double-tap Shift to remove the last point awarded.

Timers

Main Timer

  • Controlled using Start, Pause, and Reset.
  • To prevent accidental disruption during a match, the Pause, Reset, and Set Custom Time buttons must be pressed and held briefly to activate.
  • A bell sound plays automatically when time expires.

Question Timer

  • Alternate Section: 10 seconds per question
  • Buzzer Section: 5 seconds per question
  • Keyboard controls:
  • R — Start / Restart Question Timer
  • P — Pause Question Timer
  • +/- — Adding a score automatically pauses the question timer.
  • Double-tapping the Right Arrow (next visual) or Left Arrow (next audio) also automatically starts the Question Timer.

Visual Questions

  • Click Upload Visuals to add images.
  • Reorder visuals using drag-and-drop.
  • During the match, double-tap Right Arrow to display the next visual full-screen on the Main Scoreboard.
  • Visuals automatically disappear 2-3 seconds after being shown.

Audio Clips

  • Upload audio clips from the Control Panel.
  • Trim clips using the waveform slider to select the exact portion needed.
  • Reorder audio clips via drag-and-drop.
  • Match controls:
  • Double-tap Left Arrow — Play next audio clip
  • Press Down Arrow — Stop current audio

C) Presenting Mode

Presenting Mode is designed for a clean dual-screen competition setup.

  • Automatically places the Main Scoreboard on the second screen in fullscreen.
  • Automatically keeps the Control Panel on the main display.
  • Use Presenting Mode when running live matches or practice sessions on a projector/TV.

D) Live Scoreboard

The Live Scoreboard is a live preview panel inside the Control Panel that mirrors key items being shown on the audience display (scores, round/section, timers, and the occasional visuals). It helps the coach confirm everything is correct at a glance while operating controls.

  • Use it as a quick verification view while scoring and changing sections.
  • It updates as the match progresses and reflects the active section/timer state.
